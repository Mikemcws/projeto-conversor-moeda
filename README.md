# 💱 Conversor de Moeda

Aplicação web desenvolvida com HTML, CSS e JavaScript puro que permite converter valores entre diferentes moedas de forma simples e intuitiva.

## 🚀 Funcionalidades

- ✅ Converter entre 5 moedas: Real (BRL), Dólar (USD), Euro (EUR), Libra (GBP) e Bitcoin (BTC)
- ✅ Interface responsiva e moderna
- ✅ Exibição visual da moeda de origem e destino com ícones
- ✅ Validação de entrada de dados
- ✅ Resultado exibido em tempo real ao clicar em converter

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura da aplicação
- **CSS3** — estilização e layout responsivo
- **JavaScript** — lógica de conversão e manipulação do DOM

## ▶️ Como usar

1. Clone o repositório:
```bash
git clone https://github.com/Mikemcws/projeto-conversor-moeda.git
```

2. Abra o arquivo `index.html` no navegador

3. Selecione a moeda de origem, a moeda de destino, digite o valor e clique em **Converter**
